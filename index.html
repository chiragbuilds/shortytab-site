Navigate through your favorite sites with keyboard shortcuts

Shortytab transforms how you browse the web by giving you instant access to your most visited sites with a simple keyboard shortcut.

Install Extension View Source Code

Key Features

Keyboard-Centric

Access your shortcuts with Ctrl/Cmd + Shift + Y. Navigate with Y, open with release.

Beautiful UI

Enjoy a sleek glassmorphism design that blends seamlessly with any website.

Fully Customizable

Add, edit, or remove shortcuts to match your browsing habits perfectly.

Lightning Fast

Navigate to your favorite sites in seconds without touching your mouse.

No Tracking

Your data stays on your device. No analytics, no tracking, no nonsense.

Chrome Optimized

Built specifically for Chrome with modern APIs for the best performance.

Easy Installation

Install Shortytab directly from GitHub in just a few simple steps

Download

Download the ZIP file from GitHub repository

Extract

Extract the downloaded ZIP file to a folder

Extensions Page

Navigate to chrome://extensions in your browser

Developer Mode

Enable "Developer mode" in the top right corner

Load Unpacked

Click "Load unpacked" and select the extracted folder
